# BOX-MARADHI
Mfumo huu umebuniwa kuwa kama kitabu cha kidijitali ambapo kila herufi kutoka A mpaka Z inawakilisha mlango wa kipekee wa taarifa za kiafya. Pindi mtumiaji anapofika kwenye ukurasa wa mwanzo, atakutana na visanduku 26 vilivyopangiliwa kwa ustadi. Kila kisanduku kina herufi yake kuanzia A hadi Z, kikiwa na picha ya kuvutia ya herufi husika.
